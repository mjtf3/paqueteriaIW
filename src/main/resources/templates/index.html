Paqueteria IW

Bienvenido a la paqueteria IW

API de Tarifas

  • GET /api/tarifas/general - Todas las tarifas
  • GET /api/tarifas/general?tipo=distancia - Solo distancia
  • GET /api/tarifas/general?tipo=peso - Solo peso
  • GET /api/tarifas/especifica?distancia={1-4}&peso={decimal} - Calcular precio (Requiere API Key)

API de Envíos

  • POST /api/envios - Crear envío (Requiere API Key)

Ejemplo de cuerpo JSON:

{
  "direccionOrigen": "Calle Origen",
  "direccionDestino": "Calle Destino",
  "nombreComprador": "Juan Pérez",
  "peso": 2.5,
  "distancia": "NACIONAL", 
  "numeroPaquetes": 1,
  "fragil": false,
  "nota": "Entregar por la tarde"
}

Nota: El campo distancia acepta los nombres (CIUDAD, PROVINCIAL, NACIONAL, INTERNACIONAL) o sus números correspondientes (1, 2, 3, 4).
